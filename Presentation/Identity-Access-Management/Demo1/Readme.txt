Some additional points:

1-iOS code

In order to keep the size of this package small, it does not contain the binaries and code for the iOS version of this app. If you are looking for the full package, download it from here:


http://aka.ms/Co1192

Keep in mind that demoing the iOS code requires a Mac and the dev license for Apple devices

2-SharePoint and O365

In order to demo the O365 part of the code you will need an O365 subscription set. The Demo.md will show you to get it but if all you want is to demo the client app, the mobile service and authentication, you can probably remove the O365 part of it and simplify
